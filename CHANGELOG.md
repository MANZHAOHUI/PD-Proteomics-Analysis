# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-15

### Added
- Initial release of PD Proteomics Analysis pipeline
- Complete differential expression analysis pipeline in R
- Network analysis and key driver identification in Python
- MEGENA-like co-expression network construction
- Bayesian causal network inference
- Sex-stratified analysis workflows
- Comprehensive documentation and figure generation instructions
- HPC-optimized scripts for large-scale proteomics data
- Support for multiple data formats (TMT, label-free)

### Features
- Analysis of 532 PD brain samples across 3 cohorts
- Identification of 612 key driver proteins
- Module detection using multiscale clustering
- eQTL integration for causal inference
- Pathway enrichment analysis
- Publication-ready visualization functions

### Documentation
- Complete README with installation and usage instructions
- Figure generation guide with software-specific settings
- Code comments and function documentation
- Example workflows and tutorials
